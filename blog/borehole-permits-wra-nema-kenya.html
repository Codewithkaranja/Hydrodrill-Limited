Borehole Permits in Kenya (WRA & Environmental Considerations)

Before drilling, many projects require approvals. This page is a planning guide so you avoid delays, rework, or compliance issues.

Typical permit workflow (high level)

  1. Site assessment + hydrogeological survey
  2. Application documentation for water abstraction licensing
  3. Drilling + test pumping + completion report
  4. Pump installation and operation documentation

Common documents clients prepare

  • Land ownership/authority documents
  • Site location details (maps/coordinates if available)
  • Intended use: domestic / irrigation / commercial
  • Survey report + drilling completion records

Common mistakes that delay approvals

  • Skipping proper survey documentation
  • Unclear water use/abstraction estimates
  • Missing completion/testing records

FAQs

Can Hydrodrill advise on the required documentation?

Yes — we guide clients on the project documentation needed for smooth planning and execution.

Is testing important?

Test pumping and completion records help demonstrate performance and support proper system sizing.
